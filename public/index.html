Licencias — revocación y registro (ID + PC)

Contraseña = ADMIN_SECRET en Vercel. Los UUID revocados viven en Redis (y opcionalmente en REVOKED_LICENSE_IDS).

Registro en Redis: puedes guardar el License ID junto al ID de equipo (HWID) del cliente para listarlo y borrarlo todo (revocación + datos del PC) cuando quieras.

Borrar registro completo: elimina en Redis la entrada de revocado (si existía) y los datos guardados (HWID, nombre, etc.). Si rellenas HWID al borrar, debe coincidir con el guardado; si lo dejas vacío, borra sin comprobar.
